# WebRTC

A project similar to Google Meet built on the WebRTC protocol.

Project Outcomes:


- Learnt and implemented common WebRTC/related Jargon like:
  - ICE Candidates
  - STUN Servers
  - SDPs
  - NAT
  - Local Description
  - Remote Description
- Used WebSockets as a signalling server to exchange ICE Candidates
